## 课程简介  [课程大纲及资源](syllabus.md)

近年来，大型语言模型（LLM）为软件开发带来了革命性的新范式。传统的软件开发生命周期正在各个阶段被人工智能驱动的自动化技术重塑，由此引发了一个关键问题：新一代软件工程师应如何利用这些技术进步，将自身生产力提升十倍，并为未来的职业发展做好准备？

本课程将展示，现代人工智能工具不仅能够显著提升开发者的效率，还将使更广泛的群体能够参与软件工程。我们将说明，软件开发已从从零开始编写代码的模式，演变为一个包含“规划—由AI生成—修改—迭代”的循环工作流程。学生将深入掌握传统软件工程挑战背后的理论知识，同时学习当前由人工智能赋能的前沿工具如何解决这些问题。

通过实践性工程任务以及来自构建这些革命性工具的行业先驱的专题讲座，你将获得在AI辅助开发、自动化测试、智能文档生成和安全漏洞检测等方面的实战经验。完成本课程后，你将清晰理解如何将最先进的大型语言模型融入复杂的开发流程，并有效规避常见误区。

## 学分
3学分

## 先修要求
具备与CS111相当的编程基础，建议修过CS221或CS229课程。

## 授课形式
每周专题讲座、动手编程实践课，以及来自业界的特邀嘉宾分享。课程以一个体现现代开发实践的期末项目作为总结。

## 课程目标
掌握现代开发工具，深入理解AI辅助编程，学习自动化测试与部署方法，探索软件工程领域的新兴趋势。
